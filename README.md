# 📝 디지털 생활 성찰 도구

초등학생을 위한 디지털 생활 성찰 도구 - 학생들이 자신의 디지털 사용 습관을 스스로 진단하고 성찰할 수 있도록 돕는 웹 기반 활동지입니다.

## 🌐 Live Demo
👉 **[https://concept-based-edu.github.io/digital-life-reflection/](https://concept-based-edu.github.io/digital-life-reflection/)**

## 📌 프로젝트 소개

### 목적
- 초등학생들의 건강한 디지털 기기 사용 습관 형성
- 자기 주도적 디지털 생활 성찰 능력 함양
- 디지털 리터러시 교육의 실천적 도구 제공

### 주요 특징
- 🎯 **단일 페이지 웹앱**: 별도 설치 없이 브라우저에서 즉시 사용
- 📱 **반응형 디자인**: PC, 태블릿, 스마트폰 모두 지원
- 💾 **로컬 저장**: 서버 없이 브라우저에 데이터 저장
- 🖼️ **이미지 저장**: 작성한 내용을 이미지로 저장 가능
- 🎨 **친근한 UI**: 초등학생 눈높이에 맞춘 디자인

## 🚀 주요 기능

### 1. 디지털 활동 기록 (8단계)
1. **📛 나는 누구?** - 이름 또는 별명 입력
2. **🎯 오늘 사용한 것들** - 디지털 기기/앱 다중 선택
   - 스마트폰, 유튜브, 게임, SNS/채팅, 학습앱, TV, 태블릿, 컴퓨터
3. **⏰ 사용 시간** - 슬라이더로 직관적 시간 기록 (0~8시간)
4. **😊 오늘의 기분** - 6가지 감정 이모지 선택 + 이유 작성
5. **✨ 기억에 남는 것** - 서술형 작성 (도우미 버튼 제공)
6. **💭 돌아보기** - 잘한 점과 아쉬운 점 작성
7. **🎯 내일의 계획** - 내일의 나와 약속하기
8. **🤝 친구와 공유** - 친구들과 생각 나누기 (선택)

### 2. 성찰 카드 생성
- 작성한 내용을 한눈에 볼 수 있는 요약 카드
- 실시간 업데이트로 작성 내용 즉시 반영
- 진행률 표시 (0~100%)

### 3. 저장 및 공유 기능

#### 💾 브라우저 저장
- 로컬 스토리지를 활용한 데이터 저장
- 이전 기록 불러오기
- 기록 모아보기 (최근 10개 표시)

#### 📸 이미지 저장 옵션
- **교사용**: 전체 학습지 상세 내용
- **학생용**: 
  - 심플 카드 (간단한 요약)
  - 컬러풀 카드 (화려한 디자인)
- **수료증**: 진행률 80% 이상 시 발급

#### 🖨️ 인쇄
- 인쇄 최적화 CSS 적용
- 불필요한 요소 자동 숨김

## 💻 기술 스택

### Frontend
- **HTML5**: 시맨틱 마크업
- **CSS3**: 
  - Flexbox & Grid 레이아웃
  - 그라디언트 & 애니메이션
  - 반응형 미디어 쿼리
  - 인쇄 전용 스타일
- **Vanilla JavaScript**: 프레임워크 없는 순수 JS
- **html2canvas**: 이미지 캡처 라이브러리

### 저장소
- **LocalStorage**: 브라우저 로컬 저장
- **JSON**: 데이터 구조화

## 📚 사용 방법

### 학생용
1. 웹사이트 접속
2. 이름 입력 및 8단계 활동 작성
3. 성찰 카드 확인
4. 이미지로 저장 또는 인쇄
5. 선생님께 제출

### 교사용
1. 학생들에게 링크 공유
2. 활동 안내 및 지도
3. 제출된 이미지/인쇄물 확인
4. 피드백 제공

## 🌟 특별 기능

### 1. 작성 도우미
- 서술형 질문에 대한 시작 문구 제공
- 클릭 한 번으로 문장 시작 가능

### 2. 시간별 색상 변화
- 2시간 이하: 초록색 (적절함)
- 2-4시간: 주황색 (주의)
- 4시간 이상: 빨간색 (과다)

### 3. 다양한 저장 형식
- 교사 확인용 상세 보고서
- 학생 공유용 예쁜 카드
- 성취 인증 수료증
